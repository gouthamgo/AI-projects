**Customer Feedback Analyzer** is a full-stack web application that transforms customer feedback data from CSV files into actionable insights. Upload a CSV file, select a feedback column, and get detailed analytics such as sentiment distribution, feedback categories, top words, and average feedback lengthâ€”all visualized in an elegant, modern UI.

The project is built with:
- **Frontend**: Next.js for a responsive and interactive UI.
- **Backend**: FastAPI for high-performance API endpoints and NLP-based data processing.
- **Visualization**: Recharts for creating dynamic charts.


## Features

- **CSV Upload**: Upload feedback data in CSV format.
- **Column Selection**: Choose the column containing feedback text for analysis.
- **Sentiment Analysis**: Visualize sentiment distribution (positive, negative, neutral).
- **Feedback Categories**: Identify common themes (e.g., service, product) in feedback.
- **Top Words**: Discover frequently used words in feedback.
- **Average Feedback Length**: Calculate the average length of feedback entries.
- **Modern UI**: A sleek, responsive design with animations (using Framer Motion) and Tailwind CSS for styling.
- **Charts**: Interactive bar and pie charts for data visualization (using Recharts).
